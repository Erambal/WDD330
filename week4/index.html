Forms
Accessing Form Elements To get the first form object access using brackets -> const form = document.forms[0]; You can also use the name of the attribute. -> const form = document.forms['search']; You can have access to multiple things useing this -> const [input,button] = form.elements; You can also give these attributes names in you JS -> const input = form['searchInput']


Buttons Here is the code to create a button and submit -> To add an action add this form.action = '/an/other.url'


Form Events This is for you to put event listeners -> const input = form.elements.searchInput; input.addEventListener('focus', () => alert('focused'), false); To add the hover effect do this -> input.addEventListener('blur', () => alert('blurred'), false); This gives you an alert if you changed the value inside the search box. input.addEventListener('change', () => alert('changed'), false);


Submitting a Form
Add this to main.js to send the information to the server. const form = document.forms['search']; form.addEventListener ('submit', search, false); function search() { alert(' Form Submitted'); } To prevent from linking to another page do this function search(event) { alert('Form Submitted'); event.preventDefault(); }


Retrieving and Changing Values from a Form

To get a value back from the user do this. function search(event) { alert(`You Searched for: ${input.value}`); event.preventDefault(); You can also just do this. input.value = 'Search Here'; To fix the text that remains in the field when the user clicks it do this. input.addEventListener('focus', function(){ if (input.value==='Search Here') { input.value = '' } }, false); input.addEventListener('blur', function(){ if(input.value === '') { input.value = 'Search Here'; } }, false); }
