Form Validation

Here's how you manage the CSS in form validation

 
                input:invalid {
                border: 2px dashed red;
                }
    
                input:valid {
                border: 2px solid black;
                }

And when it's required add this


input:invalid:required {
background-image: linear-gradient
(to right, pink, lightgreen);
}
