.this

This command allows you to grab something specific from an object

EXAMPLE: 
            
        let user = {
            name: "John",
            age: 30,
          
            sayHi() {
              // "this" is the "current object"
              alert(this.name);
            }
          
          };
          
          user.sayHi(); // John
This will display the name of the user -> John

Objects are variables with attributes. For example: 


            const superman = {
                name: 'Superman',
                'real name': 'Clark Kent',
                height: 75,
                weight: 235,
                hero: true,
                villain: false,
                allies: ['Batman','Supergirl','Superboy'],
                fly() {
                    return 'Up, up and away!';
                }
            };
Here we created superman and then added attributes to him.

To access an attribute better called a property you use the notation you previously used. 


                superman.name 
                << 'Superman'
    
                or you could use superman['name']
                Use this when you want to add a space between the property
                
                superman["real" + " " + "name"] // the property is built using string concatenation
                << "Clark Kent"

Symbols can be used to add a property to an object 


                const name = Symbol('name');
QUESTION: Does using Symbol automatically make properties inside of it camelCase? -> 

                const realName = Symbol('real name');
                supergirl[realName] = 'Kara Danvers';
                << 'Kara Danvers'

Methods parenthesis always have to placed after method name: 


                superman.fly()
                << 'Up, up and away!'
    
                or you can use
                superman['fly']()
                << 'Up, up and away!'

Use this to check if an object has a property in it. 


        'city' in superman;
        << false
    You can also use .hasOwnProperty
    -> superman.hasOwnProperty('city');
        << false
        superman.hasOwnProperty('name');
        << true

Object.keys()

This will return an array of all the keys of the object 

        for(const key of Object.keys(superman)) {
        console.log(key);
    }
    <<  name
        real name
        height
        weight
        hero
        villain
        allies
        fly
You can also use Object .values to see the value of everything You can also use Object .entries to see both values and keys

To add a property Just assign a new property 


        superman.city = 'Metropolis';
    << 'Metropolis'

To change a property Reassign it. 


        superman['real name'] = 'Kal-El';
    << 'Kal-El'

To remove a property Use delete 


        delete superman.fly
