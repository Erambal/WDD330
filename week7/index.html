Week 7 Notes

Translation

Use this to see movement in position

translate(x,y)
        Here's an example on how to use this.
        transform: translate(45px, -45px);
        To move vertically use
        transform: translateX(45px);
        To move horizontally use:
        transform: translateY(45px);

Hint* Transforms don't work on inline elements use display: inline block;

Scaling

This makes your element grow or shrink. It also works with x and y values -> scale(x,y) Here's an example -> 

transform: translateX(40px) scale(1.5);

Rotation

Yep! This makes your css spin.

transform: rotate(10deg)

Skew

This slants the element

tranform: skew(15deg, 4deg);

I wonder if by using this you can make words vertical? Like one letter under the other.

Transform Origin

It's similar to background-position and it looks like this: 

transform-origin: 0 0;

You got to be careful using this though! Depends on where you put the translate and rotate functions varies the looks of the element For example ->

 transform: translateX(200px) rotate(135deg);

Looks different from ->

 transform: rotate(135deg) translateX(200px);

Here are the steps to create a simple transition using only CSS:

Declare the original state of the element in the default style declaration.

Declare the final state of your transitioned element; for example, a :hover state.

Include the transition functions in your default style declaration using the transition properties, including: transition-property , transition-duration , transition-timing-function , and transition-delay .

For older browsers you need to use the -webkit- prefix

Transition property

Here's how you use this ->

 transition-property: transform, color;

There's also a transition-duration

transition-duration: 0.2s;

and a transition-timing-function that lets you control the pace of the transition in even more granular detail. 

transition-timing-function: ease-out;

There's transition-delay

transition-delay: 50ms;

Animations

Use keyframes


            @keyframes moveRight { 
                from {
                transform: translateX(-50%);
                }
                to {
                transform: translateX(50%);
                }
            }
            @keyframes appearDisappear { 
                0%, 100% {
                opacity: 0;
                }
                20%, 80% {
                opacity: 1;
                }
            }
            @keyframes bgMove {
                100% {
                background-position: 120% 0;
                }
            }
